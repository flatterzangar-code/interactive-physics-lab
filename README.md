## 📱 About

**Physics Lab** is an educational mobile application developed using **Flutter**.

The app helps students understand physics using:

- 📚 Theory explanations
- 🎬 Interactive animations
- 🧠 Topic-based quizzes

Each topic follows a structured learning process:
Learn → Visualize → Test.

---

## ✨ Features

- ✅ 13 Physics Topics
- 🎬 Animated physics simulations
- 📖 Educational content
- 🧠 Integrated quizzes
- 📊 Knowledge evaluation
- ⚡ Smooth UI/UX

---

## 🧠 Learning Flow

Home → Topic → Theory → Animation → Test → Result
